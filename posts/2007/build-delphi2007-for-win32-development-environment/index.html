搭建 Delphi 2007 For Win32 开发环境

Written by Delphier with ♥ on 10 December 2007 in Delphi 1 min

RAD Studio 2007 虽然包含 Delphi For .NET、C++ Builder、ECO，但很多人使用的其实还是 Delphi For Win32，用来开发 Windows 环境下的普通桌面程序。下面是搭建最新的 Delphi 2007 在 Win32 平台下开发环境所需的组件，在这里记录一下：

下载 .NET Framework 2.0 及简体中文语言文件
下载 Visual J# 组件及简体中文语言文件
下载 .NET Framework 2.0 SDK
下载 Delphi 2007 For Win32 （434M，包含 Update 1）

按照以上顺序安装即可，记得在安装 Delphi 时，不要选择移除安装缓存。安装完成后，可用开始菜单中的 Check For Updates 将 Delphi 2007 自动升级到最新版本。

最后一点就是 Delphi 2007 破解，网上有很多方法，你可以下载 Delphi Distiller，移除许可检查即可。
