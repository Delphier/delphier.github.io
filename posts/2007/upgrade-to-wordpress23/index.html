升级到了 WordPress 2.3

Written by Delphier with ♥ on 21 October 2007 in Blog 1 min

WordPress 2.3 已经推出很长时间了，一直没有升级，之前也没有尝试过 WordPress 升级，而且此版本变化大，担心升级以后带来问题。

昨天晚上心血来潮，到 WordPress 网站看了一下，了解一下升级的方法。看完以后，发现升级非常简单，按以下步骤即可：

  • 备份。（以防万一，不是必须的）
  • 下载 WordPress 最新版，然后覆盖原来的所有文件。（这一步其实很简单，将所有文件直接上传即可，不用担心会影响配置文件和数据）
  • 执行 wp-admin/upgrade.php，升级数据库。

一般情况下，执行上述第 2 和 第 3 步即可完成 WordPress 升级。

就是这么简单，当然如果用了 WordPress 插件的话，升级完以后，需检查一下插件的运行情况或安装最新版的相关插件。

最喜欢 WordPress 2.3 的一个功能是优化了 URL，将不带 www 的地址自动跳转到 www 开头的 URL 上。
