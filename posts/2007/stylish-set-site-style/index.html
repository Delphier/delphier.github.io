将一个网站的风格设置为你喜欢的样式

Written by Delphier with ♥ on 10 September 2007 in Blog 1 min

一个网站提供的功能和服务很好，但他的界面你却不喜欢，怎么办？如果你用的浏览器是 Firefox，那就好办了，安装一个插件，通过自定义该网站的 CSS，就能使网站风格变为你喜欢的样式。

这个 Firefox 插件就是 Stylish，可以在这儿下载安装。安装后会在状态条生成一个图标，点击该图标即可自定义该网站的 CSS，可以针对网站的某一页，也可针对于网站全域，当然，前提是你必须要会写 CSS，应用你的自定义样式后，每次访问该网站都是自定义后的风格。

如果你重新安装系统，是不是要备份一下 CSS 或重新撰写？完全没必要，UserStyles.org， 就是一个 Stylish 自定义 CSS 分享站点，在这个网站上，你可以搜索别人已经制作好的 CSS，当然你也可以把你的 CSS 上传到这个网站与别人分享，当你每次重新安装 Firefox 后，就可以直接从该网站下载你的自定义样式。

OneFly 很喜欢 Todoist，但对他的字体样式和大小不是很满意，所以自定义了一个样式 My Todoist Style 来实现自己要求的效果。
