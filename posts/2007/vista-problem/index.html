用不用 Vista？这是一个问题！

Written by Delphier with ♥ on 12 October 2007 in Microsoft 2 min

昨天下午，把本本的系统换成了 Windows Vista，打算从此以后放弃 XP 了。顺便把安装和使用中的一些问题，记录一下：

  • Vista 的安装时间较 XP 明显缩短，仅仅用了 25 分钟左右系统就可用了；
  • 发现安装过程中不再提示格式化分区选项，但可以在选择安装分区对话框手动格式化；
  • Vista 在启动过程中，在屏幕中上方有时会显示一串破折线， 不知道是为什么；
  • Vista 桌面非常漂亮，图标很大，照顾了流行的大屏趋势，14.1 的 1280×800 反而显得可用空间有点小；
  • 系统字体更新为微软雅黑体，习惯以后感觉很好，大字号下很美观，不过微软雅黑的英文字母有点丑；
  • 宋体在 Vista 下非常难看，总之样子很奇怪；
  • 并不是 Vista 下的所有程序都使用雅黑体，任务管理器和 Live Messenger 等就是宋体；
  • 既然用雅黑了，为什么 IE 默认字体还是宋体呢？将 Vista 的字体使用统一一下会更好；
  • 窗口最大化以后标题栏会变黑，是非常黑的那种，感觉是个败笔，不过习惯以后也就好了；
  • 看了下 Vista 版本号，是版本 6.0 6000，Explorer.exe 版本是 6.0.6000.16386；
  • Vista 的一些中文翻译很生硬，有时不知所云，还有一些明显错误，比如断开一个宽带连接时的提示信息；
  • Vista 的功能导航设计应该说是很失败，要找到一个功能要点很多下，比 XP 繁琐；
  • 找了半天也不知道建立宽带连接在哪儿，有关网络的那几个任务选项看字面意思都差不多，只能挨个儿试一遍；
  • 有个问题报告和解决方案，会提示应该安装哪些驱动等类似问题，很不错，但每当有程序崩溃时，他也跳出来，弄出一个进度条找来找去的，有这个必要吗？
  • Windows Update 变成一个桌面程序了，非常方便，不用再到网站上去查找更新了；
  • 用户帐户控制－UAC 很麻烦，用了一段时间后就在控制面板里禁用了；
  • 在 XP 下，当一个网络受限或无连接时，可选择不通知，但 Vista 没有这个选项；
  • Vista 比 XP 上网速度慢，可能是心理作用；

用了一天，感觉 Vista 很不错，也没有网上说的那么多问题；但也不是完全没有问题，有时浏览网页时，有些网站无法打开，多次刷新也不行，不管是用 Firefox 还是 IE，但如果断开宽带以后重新连接，却又可以打开；并且 Firefox 在切换 Tab 页时，有时会有几秒的无响应，感觉“卡”住一样。

在 Google 里搜了一下，也没有找到解决方案，看来，还得换回 XP 啊！
