PHP 的 ceil() 和 floor() 函数

Written by Delphier with ♥ on 28 August 2007 in PHP 1 min

PHP 的函数是非常丰富的，比如这个 ceil() 函数，可以获得不小于传入参数的一个整数，即获得一个大于或等于参数的整数，例如：

ceil(0) = 0;
ceil(1.2) = 2;
ceil(2.5) = 3;

一般在做分页时，会计算总页数，这时用 PHP 的 ceil() 函数将非常方便：总页数 = ceil(记录总数/每页记录数) 。

与 ceil() 函数对应的是 floor() 函数，他可以获得不大于参数的一个整数，即小于或等于传入参数的整数。

关于这两个函数的详细介绍可以查看 PHP 手册中的数学函数参考。
