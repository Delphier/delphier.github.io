转换 Delphi 穷举值为字符串

Written by Delphier with ♥ on 24 September 2008 in Delphi 1 min

在运行时将一个穷举类型的元素值转换为字符串：

type
    TProgrammerType = (tpDelphi, tpCS, tpVB);

uses TypInfo;
...

var s: string;
s := GetEnumName(TypeInfo(TProgrammerType), Integer(tpDelphi)) ;
//s = 'tpDelphi'
